textCell
textCell.js is an animation script that simulates cellularity and gravitation. I have written this script to accompany a scientific essay being published in an online magazine. textCell extracts and parses text from the body of an article, and then simulates its animation using HTML Canvas. New paragraphs are animated as a user scrolls.
